Search Bar 1: For this feature of lookinit you type in a URL without https:// for example basketrandom.com. When you do this it opens a iframe in a new window where you can use the website. This feature does not work with websites with backend servers poki, discord, youtube etc.

Search Bar 2: For this feature of lookinit you search a URL and lookinit search the json file linked in the github project and shows the URL as a clickable button, this just opens the URL if the URL is blocked it does not work.

Short Cuts: Click on the symbol to quickly access sites such as youtube, gmail, discord, chatgpt, google, google docs, google sites, echo, IXL, and finally google classroom.

Form: Just fill out the form.
